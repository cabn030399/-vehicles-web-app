
# Aplicación Web de Análisis de Datos del Torneo

## Descripción del Proyecto

Esta aplicación web permite visualizar y analizar los datos del torneo, mostrando la relación entre las jornadas y el total de partidos jugados en cada una.

## Funcionalidades

La aplicación ofrece dos tipos de visualizaciones interactivas:

- **Histograma**: Muestra la distribución de partidos por jornada
- **Gráfico de Dispersión**: Visualiza la relación entre las jornadas del torneo y la cantidad de partidos

## Cómo usar la aplicación

Los usuarios pueden interactuar fácilmente con la aplicación mediante botones que generan las gráficas correspondientes al hacer clic en ellos.
